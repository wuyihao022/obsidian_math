### 关于本仓库

本仓库 Fork 自 [CanisAlpha 的考研数学笔记](https://github.com/BlandAlpha/obsidian_math)，基于 Obsidian 的 markdown 笔记系统。

**我的情况：**
- 💼 **在职考研**：在工作的同时，利用业余时间备考
- 🎯 **目标明确**：杭电或复旦的非全计算机专业
- 📚 **持续学习**：通过整理笔记，加深理解，巩固知识

本仓库理论上说可以被任何markdown阅读器打开；但Obsidian可以使用**标签**、**内部链接跳转**和**知识图谱**功能，可以渲染**CallOut块**，获得最完整的体验。

### 如何使用

本地使用只需下载本仓库并使用Obsidian打开即可。

#### Git 快捷键

1. Commit all: `Alt + C`
2. Pull: `Ctrl + Shift + -`
3. Push: `Ctrl + Shift + +`

你可以在设置中更改自己的快捷键。

### 注意事项

1. 为督促学习，每次学习后我将进行一次提交，提交备注是当前日期。若仓库进行了重大变更等，会在提交中进行额外说明。
