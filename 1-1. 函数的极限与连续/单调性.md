## 单调性

### 定义
设函数 $f(x)$ 在区间 $I$ 上有定义：
- **单调递增**：对于任意 $x_1 < x_2$，都有 $f(x_1) \leq f(x_2)$
- **严格单调递增**：对于任意 $x_1 < x_2$，都有 $f(x_1) < f(x_2)$
- **单调递减**：对于任意 $x_1 < x_2$，都有 $f(x_1) \geq f(x_2)$
- **严格单调递减**：对于任意 $x_1 < x_2$，都有 $f(x_1) > f(x_2)$

### 通俗理解
- **递增**：随着 $x$ 增大，$f(x)$ 也增大（或不变）
- **递减**：随着 $x$ 增大，$f(x)$ 减小（或不变）

### 判断方法

#### 1. 定义法（作差法）
步骤：
1. 任取 $x_1, x_2 \in I$，且 $x_1 < x_2$
2. 作差：$f(x_2) - f(x_1)$
3. 判断符号：
   - 若 $f(x_2) - f(x_1) > 0$，则函数递增
   - 若 $f(x_2) - f(x_1) < 0$，则函数递减
   - 若 $f(x_2) - f(x_1) = 0$，则函数为常数

#### 2. 定义法（作商法）
适用于 $f(x) > 0$ 或 $f(x) < 0$ 的情况：
1. 任取 $x_1, x_2 \in I$，且 $x_1 < x_2$
2. 作商：$\frac{f(x_2)}{f(x_1)}$
3. 判断：
   - 若 $\frac{f(x_2)}{f(x_1)} > 1$，则函数递增
   - 若 $\frac{f(x_2)}{f(x_1)} < 1$，则函数递减

#### 3. 导数法
- $f'(x) > 0$ → 递增；$f'(x) < 0$ → 递减
- $f'(x) \geq 0$ → 单调递增；$f'(x) \leq 0$ → 单调递减

### 例子

**用定义法证明** $f(x) = x^2$ 在 $[0, +\infty)$ 上递增：
- 任取 $x_1, x_2 \in [0, +\infty)$，且 $x_1 < x_2$
- 作差：$f(x_2) - f(x_1) = x_2^2 - x_1^2 = (x_2 - x_1)(x_2 + x_1)$
- 因为 $x_2 > x_1 \geq 0$，所以 $x_2 - x_1 > 0$，$x_2 + x_1 > 0$
- 因此 $f(x_2) - f(x_1) > 0$，即 $f(x_2) > f(x_1)$
- 所以 $f(x) = x^2$ 在 $[0, +\infty)$ 上严格递增

**其他例子**：
- $f(x) = x^2$：在 $(-\infty, 0]$ 上递减，在 $[0, +\infty)$ 上递增
- $f(x) = e^x$：在 $(-\infty, +\infty)$ 上严格递增
- $f(x) = \ln x$：在 $(0, +\infty)$ 上严格递增
