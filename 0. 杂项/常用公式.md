$$\sec x$$（正割，secant）
- $\sec x = \frac{1}{\cos x}$（余弦的倒数）
- 偶函数：$\sec(-x) = \sec x$
- 定义域：$x \neq \frac{\pi}{2} + k\pi$（$k \in \mathbb{Z}$），即 $\cos x \neq 0$ 的所有点
- 值域：$(-\infty, -1] \cup [1, +\infty)$
- 导数：$(\sec x)' = \sec x \tan x$
- 周期：$2\pi$

![sec x](assets/trig_sec.png)

$$\csc x$$（余割，cosecant）
- $\csc x = \frac{1}{\sin x}$（正弦的倒数）
- 奇函数：$\csc(-x) = -\csc x$
- 定义域：$x \neq k\pi$（$k \in \mathbb{Z}$），即 $\sin x \neq 0$ 的所有点
- 值域：$(-\infty, -1] \cup [1, +\infty)$
- 导数：$(\csc x)' = -\csc x \cot x$
- 周期：$2\pi$

![csc x](assets/trig_csc.png)

## 三角函数公式

### 基本关系式

- $\sin^2 x + \cos^2 x = 1$
- $\sec^2 x = 1 + \tan^2 x$
- $\csc^2 x = 1 + \cot^2 x$
- $\tan x = \frac{\sin x}{\cos x}$
- $\cot x = \frac{\cos x}{\sin x} = \frac{1}{\tan x}$

### 诱导公式

> **口诀：奇变偶不变，符号看象限**
> 
> - **奇变偶不变**：当角度为 $\frac{\pi}{2}$ 的奇数倍（如 $\frac{\pi}{2}$、$\frac{3\pi}{2}$）时，函数名改变（sin ↔ cos，tan ↔ cot）；当角度为 $\frac{\pi}{2}$ 的偶数倍（如 $0$、$\pi$、$2\pi$）时，函数名不变。
> - **符号看象限**：将 $\alpha$ 看作锐角，判断原函数在对应象限的符号。

#### 象限符号表

各象限中三角函数的符号：

| 象限 | 角度范围 | $\sin$ | $\cos$ | $\tan$ | $\cot$ |
|:----:|:--------:|:------:|:------:|:------:|:------:|
| 第一象限 | $0 < \alpha < \frac{\pi}{2}$ | $+$ | $+$ | $+$ | $+$ |
| 第二象限 | $\frac{\pi}{2} < \alpha < \pi$ | $+$ | $-$ | $-$ | $-$ |
| 第三象限 | $\pi < \alpha < \frac{3\pi}{2}$ | $-$ | $-$ | $+$ | $+$ |
| 第四象限 | $\frac{3\pi}{2} < \alpha < 2\pi$ | $-$ | $+$ | $-$ | $-$ |

#### 象限符号记忆法

**口诀记忆**：
- **sin**：一、二象限为正（$+$）
- **cos**：一、四象限为正（$+$）
- **tan**：一、三象限为正（$+$）

**图形记忆法（更直观）**：

1. **正弦 $\sin \theta = \frac{y}{r}$**
   - 看 **y坐标**（竖直方向）
   - 上半平面（一、二象限）$y > 0$ → $\sin$ 为正
   - 下半平面（三、四象限）$y < 0$ → $\sin$ 为负

2. **余弦 $\cos \theta = \frac{x}{r}$**
   - 看 **x坐标**（水平方向）
   - 右半平面（一、四象限）$x > 0$ → $\cos$ 为正
   - 左半平面（二、三象限）$x < 0$ → $\cos$ 为负

3. **正切 $\tan \theta = \frac{y}{x}$**
   - 看 **x和y同号还是异号**
   - 同号（一、三象限）→ $\tan$ 为正
   - 异号（二、四象限）→ $\tan$ 为负

#### 负角公式

- $\sin(-x) = -\sin x$
- $\cos(-x) = \cos x$
- $\tan(-x) = -\tan x$
- $\cot(-x) = -\cot x$

#### $\frac{\pi}{2} \pm \alpha$ 的诱导公式

- $\sin(\frac{\pi}{2} - \alpha) = \cos \alpha$
- $\cos(\frac{\pi}{2} - \alpha) = \sin \alpha$
- $\tan(\frac{\pi}{2} - \alpha) = \cot \alpha$
- $\cot(\frac{\pi}{2} - \alpha) = \tan \alpha$

- $\sin(\frac{\pi}{2} + \alpha) = \cos \alpha$
- $\cos(\frac{\pi}{2} + \alpha) = -\sin \alpha$
- $\tan(\frac{\pi}{2} + \alpha) = -\cot \alpha$
- $\cot(\frac{\pi}{2} + \alpha) = -\tan \alpha$

#### $\pi \pm \alpha$ 的诱导公式

- $\sin(\pi - \alpha) = \sin \alpha$
- $\cos(\pi - \alpha) = -\cos \alpha$
- $\tan(\pi - \alpha) = -\tan \alpha$
- $\cot(\pi - \alpha) = -\cot \alpha$

- $\sin(\pi + \alpha) = -\sin \alpha$
- $\cos(\pi + \alpha) = -\cos \alpha$
- $\tan(\pi + \alpha) = \tan \alpha$
- $\cot(\pi + \alpha) = \cot \alpha$

#### $\frac{3\pi}{2} \pm \alpha$ 的诱导公式

- $\sin(\frac{3\pi}{2} - \alpha) = -\cos \alpha$
- $\cos(\frac{3\pi}{2} - \alpha) = -\sin \alpha$
- $\tan(\frac{3\pi}{2} - \alpha) = \cot \alpha$
- $\cot(\frac{3\pi}{2} - \alpha) = \tan \alpha$

- $\sin(\frac{3\pi}{2} + \alpha) = -\cos \alpha$
- $\cos(\frac{3\pi}{2} + \alpha) = \sin \alpha$
- $\tan(\frac{3\pi}{2} + \alpha) = -\cot \alpha$
- $\cot(\frac{3\pi}{2} + \alpha) = -\tan \alpha$

#### $2\pi - \alpha$ 的诱导公式

- $\sin(2\pi - \alpha) = -\sin \alpha$
- $\cos(2\pi - \alpha) = \cos \alpha$
- $\tan(2\pi - \alpha) = -\tan \alpha$
- $\cot(2\pi - \alpha) = -\cot \alpha$

### 和差角公式

- $\sin(\alpha \pm \beta) = \sin \alpha \cos \beta \pm \cos \alpha \sin \beta$
- $\cos(\alpha \pm \beta) = \cos \alpha \cos \beta \mp \sin \alpha \sin \beta$
- $\tan(\alpha \pm \beta) = \frac{\tan \alpha \pm \tan \beta}{1 \mp \tan \alpha \tan \beta}$

### 倍角公式

- $\sin 2\alpha = 2\sin \alpha \cos \alpha$
- $\cos 2\alpha = \cos^2 \alpha - \sin^2 \alpha = 2\cos^2 \alpha - 1 = 1 - 2\sin^2 \alpha$
- $\tan 2\alpha = \frac{2\tan \alpha}{1 - \tan^2 \alpha}$

### 半角公式

- $\sin^2 \frac{\alpha}{2} = \frac{1 - \cos \alpha}{2}$
- $\cos^2 \frac{\alpha}{2} = \frac{1 + \cos \alpha}{2}$
- $\tan \frac{\alpha}{2} = \frac{\sin \alpha}{1 + \cos \alpha} = \frac{1 - \cos \alpha}{\sin \alpha}$

### 积化和差公式

- $\sin \alpha \cos \beta = \frac{1}{2}[\sin(\alpha + \beta) + \sin(\alpha - \beta)]$
- $\cos \alpha \sin \beta = \frac{1}{2}[\sin(\alpha + \beta) - \sin(\alpha - \beta)]$
- $\cos \alpha \cos \beta = \frac{1}{2}[\cos(\alpha + \beta) + \cos(\alpha - \beta)]$
- $\sin \alpha \sin \beta = -\frac{1}{2}[\cos(\alpha + \beta) - \cos(\alpha - \beta)]$

### 和差化积公式

- $\sin \alpha + \sin \beta = 2\sin \frac{\alpha + \beta}{2} \cos \frac{\alpha - \beta}{2}$
- $\sin \alpha - \sin \beta = 2\cos \frac{\alpha + \beta}{2} \sin \frac{\alpha - \beta}{2}$
- $\cos \alpha + \cos \beta = 2\cos \frac{\alpha + \beta}{2} \cos \frac{\alpha - \beta}{2}$
- $\cos \alpha - \cos \beta = -2\sin \frac{\alpha + \beta}{2} \sin \frac{\alpha - \beta}{2}$

### 万能公式

设 $t = \tan \frac{x}{2}$，则：

- $\sin x = \frac{2t}{1 + t^2}$
- $\cos x = \frac{1 - t^2}{1 + t^2}$
- $\tan x = \frac{2t}{1 - t^2}$

### 辅助角公式

- $a\sin x + b\cos x = \sqrt{a^2 + b^2} \sin(x + \varphi)$

其中 $\varphi$ 满足：$\sin \varphi = \frac{b}{\sqrt{a^2 + b^2}}$，$\cos \varphi = \frac{a}{\sqrt{a^2 + b^2}}$

或者：

- $a\sin x + b\cos x = \sqrt{a^2 + b^2} \cos(x - \theta)$

其中 $\theta$ 满足：$\sin \theta = \frac{a}{\sqrt{a^2 + b^2}}$，$\cos \theta = \frac{b}{\sqrt{a^2 + b^2}}$

### 三倍角公式

- $\sin 3\alpha = 3\sin \alpha - 4\sin^3 \alpha$
- $\cos 3\alpha = 4\cos^3 \alpha - 3\cos \alpha$
- $\tan 3\alpha = \frac{3\tan \alpha - \tan^3 \alpha}{1 - 3\tan^2 \alpha}$

## 椭圆面积公式

$$\frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} = 1$$

则其面积为： $$S = \pi a b$$

其中 $a$ 是椭圆长半轴， $b$ 是椭圆短半轴。
