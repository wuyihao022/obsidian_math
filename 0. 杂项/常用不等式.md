> [!summary] 
> 
> 考研数学中常用的不等式及其应用。掌握这些不等式对于证明题和计算题都非常重要。

## 均值不等式（算术-几何平均不等式）

### 基本形式

对于 $n$ 个正数 $a_1, a_2, \ldots, a_n$，有：

$$\frac{a_1 + a_2 + \cdots + a_n}{n} \ge \sqrt[n]{a_1 a_2 \cdots a_n}$$

当且仅当 $a_1 = a_2 = \cdots = a_n$ 时等号成立。

### 常用形式（$n=2$）

对于两个正数 $a, b > 0$：

$$\frac{a + b}{2} \ge \sqrt{ab}$$

即：

$$a + b \ge 2\sqrt{ab}$$

当且仅当 $a = b$ 时等号成立。

### 变形形式

- $a + \frac{1}{a} \ge 2$（$a > 0$）
- $a^2 + b^2 \ge 2ab$（$a, b \in \mathbb{R}$）
- $a + b \ge 2\sqrt{ab}$（$a, b > 0$）

### 应用示例

> [!example] 例：证明 $\frac{x}{1+x^2} \le \frac{1}{2}$
> 
> 由均值不等式：$|x| + \frac{1}{|x|} \ge 2$（$x \neq 0$）  
> 即：$|x|^2 + 1 \ge 2|x|$  
> 所以：$1 + x^2 \ge 2|x|$  
> 因此：$\frac{|x|}{1+x^2} \le \frac{1}{2}$

## 柯西-施瓦茨不等式（Cauchy-Schwarz）

### 基本形式

对于实数 $a_1, a_2, \ldots, a_n$ 和 $b_1, b_2, \ldots, b_n$，有：

$$(a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \ge (a_1b_1 + a_2b_2 + \cdots + a_nb_n)^2$$

当且仅当 $\frac{a_1}{b_1} = \frac{a_2}{b_2} = \cdots = \frac{a_n}{b_n}$ 时等号成立（假设 $b_i \neq 0$）。

### 常用形式（$n=2$）

$$(a^2 + b^2)(c^2 + d^2) \ge (ac + bd)^2$$

### 积分形式

$$\left(\int_a^b f(x)g(x)dx\right)^2 \le \int_a^b f^2(x)dx \cdot \int_a^b g^2(x)dx$$

## 三角不等式

### 基本形式

$$|a + b| \le |a| + |b|$$

### 推广形式

$$|a_1 + a_2 + \cdots + a_n| \le |a_1| + |a_2| + \cdots + |a_n|$$

### 反向三角不等式

$$|a - b| \ge ||a| - |b||$$

---

## 伯努利不等式（Bernoulli）

### 基本形式

对于 $x > -1$，$n \in \mathbb{N}$：

$$(1 + x)^n \ge 1 + nx$$

当 $n > 1$ 且 $x \neq 0$ 时，等号不成立。

### 推广形式

对于 $x > -1$，$\alpha \ge 1$ 或 $\alpha \le 0$：

$$(1 + x)^\alpha \ge 1 + \alpha x$$

对于 $x > -1$，$0 < \alpha < 1$：

$$(1 + x)^\alpha \le 1 + \alpha x$$

---

## 杨氏不等式（Young）

### 基本形式

对于 $a, b > 0$，$p, q > 1$ 且 $\frac{1}{p} + \frac{1}{q} = 1$：

$$ab \le \frac{a^p}{p} + \frac{b^q}{q}$$

当且仅当 $a^p = b^q$ 时等号成立。

### 常用形式（$p = q = 2$）

$$ab \le \frac{a^2 + b^2}{2}$$

---

## 切比雪夫不等式（Chebyshev）

### 基本形式

对于两组同序（都递增或都递减）的实数 $a_1 \le a_2 \le \cdots \le a_n$ 和 $b_1 \le b_2 \le \cdots \le b_n$：

$$\frac{1}{n}\sum_{i=1}^n a_i b_i \ge \left(\frac{1}{n}\sum_{i=1}^n a_i\right)\left(\frac{1}{n}\sum_{i=1}^n b_i\right)$$

对于两组反序的实数，不等号反向。

---

## 排序不等式

### 基本形式

对于两组实数 $a_1 \le a_2 \le \cdots \le a_n$ 和 $b_1 \le b_2 \le \cdots \le b_n$，有：

$$a_1b_n + a_2b_{n-1} + \cdots + a_nb_1 \le a_1b_{i_1} + a_2b_{i_2} + \cdots + a_nb_{i_n} \le a_1b_1 + a_2b_2 + \cdots + a_nb_n$$

其中 $(i_1, i_2, \ldots, i_n)$ 是 $(1, 2, \ldots, n)$ 的任意排列。

**记忆**：同序和最大，反序和最小。

---

## 琴生不等式（Jensen）

### 基本形式

如果函数 $f(x)$ 在区间 $I$ 上是**凸函数**（下凸），则对于 $x_1, x_2, \ldots, x_n \in I$ 和正数 $\lambda_1, \lambda_2, \ldots, \lambda_n$ 满足 $\sum_{i=1}^n \lambda_i = 1$，有：

$$f\left(\sum_{i=1}^n \lambda_i x_i\right) \le \sum_{i=1}^n \lambda_i f(x_i)$$

如果 $f(x)$ 是**凹函数**（上凸），则不等号反向。

### 常用形式（等权重）

$$f\left(\frac{x_1 + x_2 + \cdots + x_n}{n}\right) \le \frac{f(x_1) + f(x_2) + \cdots + f(x_n)}{n}$$

## 其他常用不等式

### 1. 平方和不等式

$$a^2 + b^2 \ge 2ab$$

$$a^2 + b^2 \ge \frac{(a+b)^2}{2}$$

### 2. 倒数不等式

对于 $a, b > 0$：

$$\frac{1}{a} + \frac{1}{b} \ge \frac{4}{a+b}$$

### 3. 幂平均不等式

对于正数 $a_1, a_2, \ldots, a_n$ 和 $r < s$：

$$M_r \le M_s$$

其中 $M_r = \left(\frac{a_1^r + a_2^r + \cdots + a_n^r}{n}\right)^{1/r}$ 是 $r$ 次幂平均。

### 4. 对数不等式

对于 $x > -1$，$x \neq 0$：

$$\frac{x}{1+x} < \ln(1+x) < x$$

对于 $0 < x < 1$：

$$x < -\ln(1-x) < \frac{x}{1-x}$$

## 应用技巧

> [!tip] 证明函数有界性的常用思路
> 
> 1. **看到有界** → 立即想到：找 $M > 0$，使得 $|f(x)| \le M$
> 2. **分析结构** → 观察函数形式，选择合适的不等式
> 3. **常用组合**：
>    - 分式形式：考虑均值不等式
>    - 平方和：考虑 $a^2 + b^2 \ge 2ab$
>    - 绝对值：考虑三角不等式

> [!important] 记忆要点
> - **均值不等式**：最常用，适用于分式、乘积等形式
> - **柯西不等式**：适用于平方和与乘积和的比较
> - **三角不等式**：适用于绝对值相关的证明
> - 记住等号成立的条件，这在证明中很重要！
